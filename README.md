## 🐣 關於這個倉庫

這裡是我每天練習程式的地方。  
我把每天學到的內容、寫過的小程式、遇到的錯誤，還有自己的練習紀錄整理在這裡。

雖然現在還在慢慢學，  
但我希望能把每天的小進步，一點一點累積起來。

> 🌱 **今天比昨天多懂一點點，就已經很棒了。**

---

## 🎀 為什麼要做這個倉庫？

因為我希望自己不是只看懂，  
而是真的有動手寫、有持續練習、有慢慢進步。

把每天的內容記錄下來，不只是整理程式碼，  
也是在幫自己留下成長的痕跡 ✨

---

## 📚 我目前在練習什麼？

目前會從基礎開始，慢慢往上學：

- 變數與資料型態
- `print()` 輸出
- `input()` 輸入
- `if / elif / else` 條件判斷
- `for` / `while` 迴圈
- 函式（function）
- 串列（list）
- 字典（dictionary）
- 錯誤排除
- 小練習題

之後希望可以慢慢加入：

- 檔案讀寫
- 小工具程式
- 小遊戲
- 自動化程式
- 更完整的小作品

---

## 🧸 我的學習方式

我現在的練習方式大概是這樣：

- 先看懂今天要學的內容
- 自己重新打一遍
- 自己試著改一些小地方
- 寫錯沒關係，先看錯在哪裡
- 慢慢練習讓自己更熟

我想讓自己養成一個習慣：

**每天寫一點，不中斷，比一次衝很多更重要。**

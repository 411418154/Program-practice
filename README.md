## 🐣 關於這個倉庫

這裡是我每天練習程式的地方。  
我把每天學到的內容、寫過的小程式、遇到的錯誤，還有自己的練習紀錄整理在這裡。

目前我主要在練習：

- Python
- C#
- 基礎程式邏輯
- 每日小練習

雖然現在還在慢慢學，  
但我希望能把每天的小進步，一點一點累積起來。

> 🌱 **今天比昨天多懂一點點，就已經很棒了。**

## 🧸 我的學習方式

我現在的學習方式大概是這樣：

- 先看懂今天要學的內容
- 自己重新打一遍
- 自己試著改一些小地方
- 寫錯沒關係，先看錯在哪裡
- 慢慢練習讓自己更熟

我想讓自己養成一個習慣：

**每天寫一點，不中斷，比一次衝很多更重要。**
